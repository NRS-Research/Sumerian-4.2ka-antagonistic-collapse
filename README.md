Repository overview: research background, dataset summary, quick start guide

Replication data and code for "Antagonistic Adaptation and System Collapse"
This repository provides the complete reproducible dataset and numerical simulation code for the conflict multiplier model, which explains the centennial-scale lag between climatic stress peaks and societal collapse across Bronze Age civilizations during the 4.2 ka event. The Sumerian civilization is used as the core quantitative case study, with four additional civilizations included for cross-validation.
---
Research Background
The 4.2 ka event (ca. 2310–2020 BCE) was one of the most severe global aridification episodes of the Holocene. Archaeological records consistently show that major Bronze Age civilizations — Sumer, Old Kingdom Egypt, the Indus Valley, and Minoan Crete — all collapsed roughly 100–200 years after the climatic aridity peak, rather than immediately.
This study proposes a conflict multiplier model to explain this lag phenomenon through a three-stage dynamical sequence:
Pressure buildup: Climatic stress gradually erodes productive capacity
Antagonistic amplification: Rising resource competition drives internal militarization and inter-polity conflict
Resilience depletion: Systemic resilience decays exponentially until collapse
The model is formalized as a first-order linear differential equation:
$\frac{dW}{dt} = r - \Gamma \cdot s(t) \cdot W$
where $W(t)$ is systemic resilience surplus, $r$ is the baseline growth rate, $\Gamma$ is the conflict multiplier, and $s(t)$ is normalized climatic pressure.

Repository Contents
Directory	Description
`data/`	Normalized time-series datasets and raw proxy records
`code/`	Python scripts for model simulation, index calculation, and sensitivity analysis
`docs/`	Detailed documentation on data construction, source tracing, and reliability verification
`figures/`	Publication-quality figures generated by the scripts
All data are aggregated into 50-year time slices covering 2500–1900 BCE and normalized to the [0, 1] interval via min-max scaling, consistent with the ±30-year dating uncertainty standard for Bronze Age archaeological contexts.

Quick Start
Requirements
Python 3.8 or higher
Required libraries: `numpy`, `scipy`, `matplotlib`
Install dependencies:
```bash
pip install numpy scipy matplotlib
```
Run the Core Simulation
```bash
# Navigate to code directory
cd code

# Run the conflict multiplier model simulation
python conflict\_multiplier\_simulation.py
```
The script will automatically:
Solve the core differential equation with baseline parameters
Compute key dynamical indicators (characteristic decay time τ, steady-state resilience W*, peak of antagonistic investment)
Generate the 3-panel simulation figure matching the main text Fig. 1
Export full simulation results to `data/conflict\_multiplier\_simulation\_results.csv`
Additional Scripts
```bash
# Calculate climate stress index from raw proxy data
python calc\_stress\_index.py

# Run parameter robustness and sensitivity analysis
python sensitivity\_analysis.py

# Generate standalone s(t) time-series figure
python plot\_stress\_index.py
```

Data Description
Core Sumerian Datasets
Index	Symbol	Proxy Indicator	Primary Source	Key Node
Climate Stress Index	$s(t)$	Speleothem Mg/Ca and δ¹⁸O (Gol-e-Zard Cave)	Carolin et al. (2019)	Peak aridity: ~2190 BCE
Antagonistic Investment Index	$\Gamma(t)$	Frequency of militarized inter-city conflicts	Cooper (1983)	Peak conflict: ~2130 BCE
System Resilience Index	$W(t)$	Total settled area of major urban centers	Adams (1981); Wilkinson et al. (2013)	Collapse baseline: ~2070 BCE
Cross-Civilization Validation
Chronological data for Old Kingdom Egypt, Indus Valley, Minoan Crete, and the Central Plains Longshan culture are compiled directly from peer-reviewed authoritative syntheses. No independent time-series reanalysis is performed. Full details are available in `docs/Cross-Civilization Chronology Notes for Table S4.md`.
