Raw Conflict Counts and Systemic Resilience Raw Data
Sumerian Civilization, 2500ĻC1900 BCE
Document Overview
This file provides the raw, unnormalized data underlying the Antagonistic Investment Index  and Systemic Resilience Index  for the Sumerian core case study. All data are aggregated into 50-year time slices following the chronological binning protocol described in Supplementary Section S2.2.3. No manual interpolation was applied to fill empty time windows; all values are derived directly from published peer-reviewed archaeological sources.

1. Raw Conflict Counts (Antagonistic Investment Index )
Inclusion criteria: Formal military conflicts between sovereign Sumerian city-states (border skirmishes, sieges, royal campaigns) attested in royal inscriptions, year-names and administrative archives. Isolated raids, legendary accounts and non-Sumerian external invasions are excluded.
